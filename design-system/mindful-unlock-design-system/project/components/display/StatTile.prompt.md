**StatTile** — compact metric cell (amber icon, big number, caption). Lay three across in a flex row with `gap: 10px`.

```jsx
<StatTile icon={<Feather name="unlock" />} value="12" label="Total Unlocks" />
<StatTile icon={<Feather name="clock" />} value="4" label="With Delay" />
<StatTile icon={<Feather name="watch" />} value="3h" label="Time Tracked" />
```
