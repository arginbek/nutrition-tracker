**Field** — the ritual reflection input: card-colored box, hairline border, multiline by default.

```jsx
<Field placeholder="I want to check..." />
<Field accent="var(--mu-purple)" placeholder="I swear this is essential because..." />
<Field multiline={false} placeholder="App name" />
```

Tint the border purple (`accent="var(--mu-purple)"`) on oath/commitment steps.
